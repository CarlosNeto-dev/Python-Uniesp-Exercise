# Exercício de Coleta de Dados em Python

Este repositório contém um script Python simples projetado para coletar dados de pessoas, como nome, gênero e altura, e então calcular e exibir algumas estatísticas básicas com base nessas informações.

## 🇧🇷 Versão em Português

### Descrição do Programa

O programa solicita interativamente que o usuário insira o nome, gênero (M/F) e altura de até 15 pessoas. Após a inserção de pelo menos 3 pessoas, o programa exibe as seguintes estatísticas e encerra a execução:

- A maior altura registrada entre todas as pessoas.
- A média da altura das pessoas do sexo masculino.
- O número total de pessoas do sexo feminino.

O programa também inclui tratamento de erros para entradas inválidas e para divisão por zero, caso nenhuma pessoa do sexo masculino seja inserida.

### Como Executar

1.  **Pré-requisitos**: Certifique-se de ter o [Python 3](https://www.python.org/downloads/) instalado em seu sistema.

2.  **Clone o repositório** (ou baixe os arquivos):
    ```bash
    git clone <URL_DO_REPOSITORIO>
    ```

3.  **Navegue até o diretório** e execute o script:
    ```bash
    cd exerciting-python-uniesp
    python exercise/exercise.py
    ```

4.  Siga as instruções no terminal para inserir os dados.

## 🇬🇧 English Version

### Program Description

The program interactively prompts the user to enter the name, gender (M/F), and height for up to 15 people. After at least 3 people have been entered, the program displays the following statistics and then terminates:

- The highest height recorded among all individuals.
- The average height of male individuals.
- The total number of female individuals.

The program also includes error handling for invalid inputs and for division by zero, in case no male individuals are entered.

### How to Run

1.  **Prerequisites**: Ensure you have [Python 3](https://www.python.org/downloads/) installed on your system.

2.  **Clone the repository** (or download the files):
    ```bash
    git clone <REPOSITORY_URL>
    ```

3.  **Navigate to the directory** and run the script:
    ```bash
    cd exerciting-python-uniesp
    python exercise/exercise.py
    ```

4.  Follow the on-screen prompts to enter the data.
